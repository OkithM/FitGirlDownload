
# Fitgirl Download

This script automates downloading files from a list of **FuckingFast** URLs. It reads a links.txt file, scrapes each page for a download link, and opens it in your browser. It pauses every 5 downloads to prevent your browser from being overwhelmed.
## Installation

1. **Install Python**

2. **Install Dependencies :**

```bash
pip install beautifulsoup4 requests
```

3. **Download the files :**
- <>Code &nbsp; >>> &nbsp; Download ZIP

4. **Create links.txt :** Add each URL on a new line.

```bash
https://example.com/page1
https://example.com/page2
```

5. **Run the script :**

```bash
python FF.py
```
